--------------------------------------------
Sources for Urban Nature Access example data
--------------------------------------------

Land use/land cover (LULC_NLCD_2021.tif): United States National Land Cover Dataset (NLCD) 2021
https://www.usgs.gov/centers/eros/science/national-land-cover-database

Population Raster (total_population_census_2010.tif) is from United States Census data, year 2010.

Area Of Interest (AOI_admin_boundaries_census_tracts.shp): Census tract polygons from the US Census Tiger data.

Values in the LULC Attribute Table (LULC_attribute_table_UNA.csv) were taken from previous sample data, whose sources are undocumented. So do not rely on them for your study.

The other inputs defined in the Workbench parameter file invest_urban_nature_access_args_MN.json (Demand Per Capita, Search Radius Mode, Decay Function, and Uniform Search Radius) represent a simple, hypothetical example with Uniform search radius.
